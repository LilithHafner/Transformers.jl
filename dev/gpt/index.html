(gpt::Gpt)(x::T, mask=nothing; all::Bool=false)

eval the gpt layer on input x. If length mask is given (in shape (1, seqlen, batchsize)), mask the attention with mask. Moreover, set all to true to get all outputs of each transformer layer.
